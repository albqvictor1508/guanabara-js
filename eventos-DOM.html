Funções

Conjunto de linhas de código que só serão executadas quando o evento ocorrer

(este conjunto de linhas são denominados blocos)

Estes são os seguintes passos para se usar uma function

  • um bloco em JS é delimitado entre {} (tipo o css)
  • precisa ser nomeado como function + ação()
  • também pode por parâmetros na function (function ação(parametro) { })
