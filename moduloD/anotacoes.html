antes de falar de condições, é muito importante citar as sequências

são as ordens(comandos) em um fluxo para gerar um script, ou seja, a sequência de instruções para fazer a atividade

porém existem situações em que a sequência precisa se comportar de duas maneiras diferentes para criar uma atividade, exemplo:

em casos de uma escolha (condição), em que dois eventos diferentes acontecerão de acordo com sua escolha, há uma diferença na sequência.

Condições

imagine a condição como uma placa que sinaliza que só pode virar à esquerda ou à direita, que te força a uma escolha, tendo 1 resultado para cada escolha.

  • Condição Simples (if)
  • Condição Composta(if e else)
  • Condição Aninhada (elseif)
  • Condições Múltiplas(switch case)

if = se

else = senão

                    
                        if (idade < 16) {
                            console.log(`tu é de menor otaro.`)
                        
                        }

essa é a estrutura da condição simples (somente if) 

                    
                        if (idade < 16) {
                            console.log(`tu é de menor otaro.`)
                        
                        }
                        else {
                            console.log(`é de maior`)
                        }

essa é a estrutura da condição composta

                    
                        if () {

                        }
                        else {

                        }
                        else if() {

                        }

essa é a estrutura da condição aninhada(pode ter quantos else if você quiser na condição)

                    
                        switch (expressão) {

                            case "valor 1":
                               (break)

                            case "valor 2":
                                (break)

                            case "valor 3":
                                (break)

                            default:
                               (break)
                        }

essa é a condição multipla

o "default" é igual o else, se n for nenhum dos outros casos, o default é ativado

o break é aonde delimita o final do case, sem o break, o comando vai executar todos os cases até achar um break

no switch não pode ser usado operadores, ou seja, não se pode criar cases com contas, ou com comparações, apenas com dados pontuais
