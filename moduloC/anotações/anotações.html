método 'window'

window.alert (emite um alerta no navegador)

window.confirm (emite um alerta no qual o usuário pode aceitar ou cancelar)

window.prompt (emite um alerta com uma caixa de texto para que o usuário possa respoonder)

window.document.write(window.document.charset) exibe no site o charset utilizado no HTML (UTF-8)

window.document.write(window.navigator.appName) exibe no site qual navegador está sendo utilizado

window.document.write(window.document.URL) exibe no site o URL
