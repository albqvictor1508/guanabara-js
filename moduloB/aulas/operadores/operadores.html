Operadores-JS

Trabalharemos com os seguintes operadores

  • Op. aritméticos
  • Op. de atribuição
  • Op. relacionais
  • Op. lógico
  • Op. ternário

Aritméticos

Antes de começar, é importante lembrar a precedência aritmética

                []
                ()
                **
                *, /, % 
                +, -

(* / %) quem aparecer primeiro da esquerda para direita, é o primeiro a ser executado

  • + = soma
  • "-" = subtrai
  • * = multiplica
  • / = divide
  • % = resto da divisão inteira
  • ** = potência

Auto-Atribuição

                
                    var n = 3
                    n = n + 4
                    n = n - 5

dps da soma "n + 4", o "n" deixa de ser 3 e passa a valer 7, depois da subtração de "n - 5", o n passa a ser 2

o mesmo se aplica às outras operações

                
                    n = n * 4
                    n = n / 2
                    n = n ** 2
                    n = n % 5

Simplificando

                
                    n += 4
                    n += 5
                    n *= 4
                    n /= 2
                    n **= 2
                    n %= 5

Incremento

uma forma de resumir a auto-atribuição(n -= 1 e n += 1)

No lugar de (n -= 1 e n += 1), fica (n++ e n--)

no NodeJS, se colocar (n++ e n--) vai retornar o valor antigo, mas se perguntar o valor da variável, dirá qual é o novo valor

Isso pode ser revertido mudando a ordem, colocando (++n e --n) no lugar de (n++ e n--) o valor será retornado no momento em que a incrementação for feita

Op. relacionais

                
                    > (maior que)
                    < (menor que)
                    >= (maior ou igual)
                    <= (menor ou igual)
                    == (igual)
                    != (diferente)

o op. relacional do JS não se limita ao tipo de dado, ou seja, um 5 e um "5" tem a mesma grandeza 

                
                    5 == "5"
                    true

Identidade (===)

serve para relacionar se dois valores são iguais em grandeza e em tipo

                
                    5 === "5"
                    false

nesse caso mostra que não são idênticos

Desigual Restrito(!==), são tipos diferentes, porém o mesmo valor

Operações Lógicas

                ! (negação)
                && (conjunção)
                || (disjunção)

Negação só tem um operando (true, false)

Conjunção é um operador binário, mas só é satisfeito tendo todas suas opções como verdadeiras, mais de um operando

Disjunção também é binário, porém basta 1 ser verdadeiro para que o resultado seja verdadeiro, diferente da Conjunção que necessita que os dois sejam verdadeiros.

Exemplos:

    idade >= 15 && idade <= 18 'a idade está entre 15 e 18?'

    estado == 'RJ' || estado == 'Mora no RJ ou em SP?'

    salario > 1500 && sexo !='M' 'o salário é maior que 1500 e o sexo é F?'

Ternário

? (interrogação) e : (dois pontos)

Funciona como uma condição (a interrogação acompanha a pergunta e os dois pontos as condições)

            teste ? true : false

essa condição será ativada com a comparação do valor guardado na variável com qualquer outro valor

Precedência

Primeiro vem as aritméticas 

            []
            ()
            **
            *, /, % 
            +, -

Depois as relacionais (leitura da esquerda para direita, quem aparecer primeiro será executado)

Depois os lógicos (também chamados de não, e, ou) 

            ! (negação)
            && (conjunção)
            || (disjunção)
